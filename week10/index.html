MDN: Validating forms:

  • Important to have required on important fields.
  • Used to make sure a information is in the correct format like date, phone number, email.

  • Able to check wether a email has already been used.

  • New passwords can require min number and specific characters to increase security.

  • Also prohibit specific characters from being used.

  • Able to limit the maximum characters used.

MDN: Using Fetch:

  • Fetch is a way to retrieve files

  • Is used to post json data.

  • Can be used to upload a file.

  • Can be used to upload multiple files.

  • syntax fetch('http://example.com/movies.json')
    .then(response => response.json())
    .then(data => console.log(data));.

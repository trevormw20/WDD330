Week 8 Notes:

Transforms and Tansitions:

  • Transform is used on older browser while transition is used on newer browsers.

  • They can effect scale, rotation, skew, opacity, oriantation.

  • This can be turned into an animation using the transition-duration:. To cause the change to take place over a set period of time.

  • Delays can be added to pause the animation/duration.

  • Multiple transitions can occur at once.

  • Keyframes can be used to make something transition at a specific key in the frame of the whole animation. By using % of completion.

  • Animations can be set to how many times to repeat or just infinite to last forever.

  • Delay can be added between new animations to cause a short pause between animations.

Canvas, SVG, Drag and Drop:

  • Canvas is the area in which a video or image is held.

  • Specific canvas size needs to be defined seperate from the overall size because it has a default size of 300 by 150.

  • Canvas can let videos have a seperate play or other functionality or have none at all. Forcing a video to keep playing.

  • Text can be added to canvas as well.

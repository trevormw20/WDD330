OBJECTS

Objects are like multi-dimensional arrays but with specific names given to each location in the array.

Var classes = {elective: “Woodshop”, science: “Chemistry”, math: “algebra 2”};

These can be accessed with classes.science or classes[“science”];

Objects allow things to be sorted easier and easy to read and locate for future use.


DOM

DOM allows you to access any html element on a page. This enables you to edit, remove or add a new element to any part of your page.

document.getElementById(exampleId);

If an element does not have a specific class or id you can always access the parent element and then use innerHTML to access the lower element.

DOM is not only good for changing but also for selecting all elements. This can be used as a filter or search function. By selecting all elements that are associated with the filter request and only showing those specific parts of the website that are associated with the element.


EVENTS

Can replace buttons with onclick to trigger a javascript function or do some javascript right in the html file.

Pictures and slides can be changed with onmouseover and onmouseout.

Common events include

  • Onchange
  • Onclick
  • Onload
  • Onmouseout
  • onmouseover
  • Onkeydown

There are events for almost any situation on a website.

Events can help create interaction and verify user input before submission.

Events listen for a change then the DOM can change the specific element to react to the event that just occurred.
